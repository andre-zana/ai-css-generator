# 🧠 Gerador de CSS com IA

Aplicação web que utiliza Inteligência Artificial para gerar código HTML e CSS automaticamente com base na entrada do usuário, exibindo o resultado em tempo real.

## 🚀 Funcionalidades

- Geração automática de HTML + CSS com IA
- Visualização em tempo real (preview)
- Interface simples e intuitiva
- Uso de prompt estruturado para respostas consistentes

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
- Fetch API
- API de IA (compatível com OpenAI / Groq)

## 📸 Como funciona

1. O usuário digita o que deseja (ex: "botão com animação")
2. A aplicação envia o pedido para a IA
3. A IA retorna o código HTML + CSS
4. O código é exibido e renderizado na tela

## ⚠️ Atenção

Este projeto NÃO deve expor a chave da API no frontend.  
Para uso em produção, utilize um backend ou variáveis de ambiente (.env).

## 🔐 Segurança

Nunca exponha sua API KEY em projetos públicos.
